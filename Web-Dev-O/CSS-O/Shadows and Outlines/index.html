Shadows and Outlines

Box shadow

Explanation:

Box shadow is a property that allows you to add a shadow effect to an element. Its syntax is like : box-shadow: h-offset v-offset blur spread color inset; here: h-offset:Defines shadow's horizontal position.v-offset:Defines shadow's vertical positionblur:Blurs the shadow.The larger the value higher the blur.spread:Expands or shrinks the shadow sizecolor:Defines the color for the shadow.inset/offset:It makes shadow inner or outer.

Best practice:

Best practice is to use the browsers in built toolkit to select the shadow or modifying it, it will make the process much easier and you'll be able to select the best shadow for your needs.To understand how to use them you can watch this video.

Text-shadow:

Explanation:

Its just like box-shadow but the difference is that it applies to text instead of the container.It works the same as box-shadow, same syntax.You just need to change the name to text-shadow from box-shadow.You can also use web toolkit for making a appealing text-shadow.

Outline:

Explanation:

An outline is a line that is drawn around elements, outside the borders, to make the element "stand out". It's commonly used for accessibility purposes, like highlighting focused elements.The syntax of outline is same as the border.It even looks quite same but they both have some differences.

Difference between border and outline:

Outline Border
  1. It does not take up space in the layout.
  2. It does not affect the total dimensions of the element. Adding an outline will not change the element's width or height.
  3. It's positioned outside the element's border edge.
  4. It is not fixed like border you can make it appear far or very close to the content using {outline-offset} property.
  1. It takes up space around the element's content and padding.
  2. It affects the total dimensions of the element. Adding a border will increase the overall width and height of the element (unless you use box-sizing: border-box;).
  3. It's positioned inside the element's outer edge.
  4. It has a fixed position after the padding and cannot be moved further or close.
Usecases:Use outline for temporary, non-layout-altering visual highlighting, primarily for focus states and accessibility. Usecases:Use border for structural and decorative element boundaries that are a permanent part of the design and layout.
