Overflow Property

Explanation

CSS over flow property is used in cases when the text or size of a container is too large than the width and height of the container that it overflows out of the container.Using this property we can handle overflowing of things inside a container.Overflow property has multiple key values which have different funtions.Here are some of the key values descriptively explained below.
Firstly the default value for overflow property is visible explained below:

1: Visible(default)

Explanation:

This property makes the whole text in the container visible irrespective of the size of the container meaning the text will overflow if its too large for the width and height of the container.

Example:

Here the text in the div is two large for the width and height of the container to contain therefore it overflows and as the overflow property is set to visible therefore the overflow shows.

2: Hidden and clip

Explanation(hidden):

This value of overflow property makes the text hidden if it exceeds the width and height of the container.In this case the text will disappear and wont show.

Overflow-x:

This sets a horizontal scroll bar if the text is scrolling horizontally, you can also set other values on it as well like (visible,hidden,scroll and auto).

Overflow-y:

This sets a vertical scroll bar if the text is scrolling horizontally, you can also set other values on it as well like (visible,hidden,scroll and auto).

Additional info:

We can write them together as: overflow: x y;.Where x is the value for overflow-x and y is value for overflow-y.

Note:

If you use any scroll value then a scroll bar will appear regardless of the text overflowing.
