Fonts in CSS

Important font properties :

○ Font size :

The font size property sets the size of the fonts. It has some predefined sizes, such as small, medium, large, larger, etc.The most commonly used units for font size are px (pixels), em (ems), rem (root ems), and percentage (%).

○ Font styles :

The font style property sets the style of the font, such as normal, italic, oblique, or inherit.

○ Quick Notes :

Normal is the default font style.Italic is used to make the text appear italic.Oblique is similar to italic but has less bend.Inherit is used when you want to use the same font style as the parent element, like if i assigned a to div then all the elements inside div will also have same font style.Inherit can also be used with other properties like color, background-color, etc.

○ Font weight :

The font weight property defines the boldness of the text.It ranges from 100(thin) to 900(super bold).It also contains predefined values such as bold,bolder and lighter.

○ Font family :

The font-family property specifies the font for an element. The font-family property can hold several font names as a "fallback" system. If the browser does not support the first font, it tries the next font.Each font is separated by commas priority wise.

Generic font family :

1- serif: generic serif fonts (like Times New Roman).

2- sans-serif: generic sans-serif fonts (like Arial or Helvetica).

3- monospace: generic monospaced fonts (like Courier New).

4- cursive: generic cursive fonts (for handwritten styles).

5- fantasy: generic fantasy fonts (for decorative styles).

Tip: It is recommended to end the font family with any of these generic font family names.
