Text in CSS

Important font properties :

○ Text decoration :

The text decoration property is used to decorate text.There are 4 types of text decorations as :

types of text decorations :

Underline: It is used to insert a line below the text.

Overline: It is used to insert a line above the text.

Line-through: It is used to insert a line on the text.

None: It is used to remove all text decorations.

○ Text alignment :

The text align property is used to align text in a container such as div etc. It is important to use a container while using this property because without a specific block to align the text the property is useless.

Values for text-alignment :

Right: It is used to align the text to the right side.

Left: It is used to align the text to the left side.

Center: It is used to center the text.

Justify: It is used to spread the text evenly between the left and right margins.

○ Text transform :

This property is used to transform the text to different cases.There are 4 values for text transform property.

Values for text-transform :

Uppercase: Transforms all letters of the text to capital letters.

lowercase: Transforms all letters of the text to small letters.

Capitalize: Transforms first letter of every word of the text to capital letter.

None: Removes any text transformation.

Examples :

○ Line-height :

Line height property is used to control the space between two lines of text.If you think we can space two different texts using margins but using margins for spacing has a problem.If you use margins for this purpose than the space you assigned will always remain fixed no matter what device screen size is or what the user's zoom settings of browser is and this may lead to inconveniences like the text appearing in inconsistent manner etc but this is not the case for line-height as it auto adjusts it for the device or specific zoom settings.
