Sizing units in css

Size in pixels

Explanation:

px (pixels) is an absolute unit of measurement in CSS. It represents a fixed number of physical pixels on thescreen. One pixel is typically the smallest visual unit on a display. However, the actual physical size of a pixel can vary depending on the screen's resolution (DPI/PPI).

Example:

Suppose i want the font style of this paragraph to be 25px then i can do that using font-size:25px.You can see its effects right now as the font size is now 25px

Size in vh and vw

Explanation:

vh (viewport height) and vw (viewport width) are relative units of measurement in CSS. They represent a percentage of the viewport's height and width, respectively. The viewport is the visible area of a web page, which may vary depending on the device's screen size and orientation.

The vh and vw units adjust the height and width of elements relative to the viewport size. This means that the dimensions of elements using these units will automatically scale based on the size of the device's screen. Unlike px, which is a fixed unit, vh and vw provide a more responsive design, ensuring that elements look consistent across different devices and screen sizes.

Example:

Suppose i want the height of this paragraph to be 20% of the viewport height then i can do that using height:20vh.You can see its effects right now as the height of this paragraph is now 20% of the viewport height

Size in em and rem

Explanation:

○ em:

The em (em) unit is a relative unit of measurement in CSS. It represents a percentage of the font size of the parent element.Simply said it is used to multiply the font size of the parent element by a number assigned by you increasing the font size of the child element by that number.The default font size of the parent element is 16px,So if i use em:2 then the font size of the child element will be 32px.Which is 2 times the font size of the parent element(2x16=32).

○ rem:

The rem (rem) unit is relative unit if measurements in CSS.It represents a percentage of the font size of the root elements(html,head,body).The default font size of the root element is 16px,So if i use rem:2 then the font size of the child element will be 32px.Which is 2 times the font size of the root element(2x16=32).

Note:

If there is no root element then rem will be same as em.Lets say that you have assigned 12px to a root element and used 2em inside the root element without assigning any font size to the child element then the font size of the child element will be 24px.Which is 2 times the font size of the root element(2x12=24) because the font size of root element will be inherited to rest of the elements.

vmin and vmax

Explanation

○ vmin:

The vmin (vmin) unit is a relative unit of measurement in CSS. It represents the smaller of the viewport's height and width. The vmin unit is useful for creating elements that scale based on the smaller dimension of the viewport, ensuring that they fit within the visible area of the screen regardless of its orientation or aspect ratio.

Example:

Here is a example of vmin taking effect, which is going to assign the width of this paragraph to 50 percent of whichever is the smaller dimension of the viewport.

○ vmax:

The vmax (vmax) unit is a relative unit of measurement in CSS. It represents the larger of the viewport's height and width.

Usecase: Imagine you want a background image or a container to always cover at least one full viewport dimension, and you want it to be based on whichever dimension is larger. vmax is ideal for this.

Example:

Here is a example of vmax taking effect, which is going to assign the width of this paragraph to 50 percent of whichever is the larger dimension of the viewport.

Percentage

Explanation:

Percentage unit in css is just like em and rem but the main difference is that while rem and em are only applicable for font, percentage unit is applicable to other sizing properties.
