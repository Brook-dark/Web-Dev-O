List of selector types:

  1. Universal Selectors
  2. Element Selectors
  3. ID Selectors
  4. Class Selectors
  5. Group Selectors
  6. Pseudo Selectors

Universal Selectors :

Universal selector is a type of selector that targets all the elements in the code meaning that any property applied to it will take effect on all the elements present in the code, see its syntax in the style.css file !

Element Selectors :

Element selector is a type of selector in which we style only a specific selector rather than the whole html webpage like we did in universal selector, properties can be applied to it using the respected element name as i am going to apply a background color 'aquamarine' to 'h1' so that every h1 tag in this html file will be affected by this and no other tag except h1 will get affected !

ID Selectors :

An ID selector is unique selector that specifies css effects to a specific html element associated with the respected ID only.Also one ID can only be used for one tag more than one tags cant have same ID, due to the ID selector this H1's background color will differ from the rest despite the element selector's properties.

Class Selectors :

Class selector is just like ID selector but it can be used to target multiple specifc elements rather than just a single element like in ID selector.

Pseudo classes

Definition: CSS rules that target an element based on its state or condition. Explanation: Imagine you have a button. A pseudo-class allows you to apply specific styles to that button only when certain conditions are met. For example, the :hover pseudo-class applies styles when the mouse hovers over the button, making it appear different to the user.
