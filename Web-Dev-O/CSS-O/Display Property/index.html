Display Property in CSS

Explanation

The display property of css is used to modify how an element will be displayed like we can change a element to an inline element or a block level element using this property despite the orginal types.Also we can align elements using this.We can use flex.We also make certain text disappear and much more.Following are some of the most important and used values for this property.

Display Inline

Explanation:

As the name says this value of display property converts an element's type to inline regardless of its orginal.Note: Setting other dimension properties like width, height, margin or padding is not allowed in Inline Display, means that any of these properties wont take effect on inline type elements or elements made inline using this value of display property.

Example:

This is div 1
This is div 2

Here both the divs are block level elements but as i used display:inline; the divs became inline and are displayed side by side.

Display Inline-Block

Explanation:

As I said before using the inline value the sizing properties wont take effect but if we use inline-block those properties will also take effect.

Example:

This is div 1
This is div 2

Here i applied some padding and margin to these divs and it applied to them despite them being inline.

Display Block

Explanation:

As the name says this value of display property converts an element's type to block regardless of its orginal.

Example:

This is span 1 This is span 2

Here both the spans are inline elements but as i used display:block; the spans became block level elements and are displayed one below the other.

Display None and visibility(hidden)

None:

Explanation:

This value of display property is used to make an element disappear from the page.

Example:

This is div 1
This is div 2

Here both the divs are displayed but as i used display:none; the divs disappeared from the page.

visibility(hidden):

Explanation:

To make a certain element hidden means it will not show but will take the space we use visibility property visibility:hidden;

Example:

This is div 1
This is div 2

Here the divs are hidden but they are taking their respective spaces.
