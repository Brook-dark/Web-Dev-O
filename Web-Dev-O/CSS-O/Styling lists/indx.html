List-Styling

Explanation:

We can style lists using various css list properties.Some of the most used properties are explained below.We can also use a single list-style property to assign multiple properties such as type,positon,image etc.

Un-ordered list styling:

Explanation:

We can style unordered list's markers using list-style-type: value; .The values can be: disc(default):A filled circle marker.circle:Hollow circle marker.square:A filled square marker.none:Removes the marker.

Ordered list styling:

Explanation:

We can also style markers of ordered lists using the list-style-type: value;The values can be: decimal(default):Decimal numbers [1,2,3...etc].decimal-leading-zero:Decimal numbers starting with '0'[01,02,03...etc].Lower-roman:Lower-case roman letters [i,ii,ii...etc].Upper-roman:Upper-case roman letter [I,II,II....etc].Lower-alpha:lower-case alphabetical letters [a,b,c...etc].Upper-alpha:Upper-case alphabetical letters [A,B,C....etc].

List's position styling:

Explanation:

The list-style-position property determines where the markers[bullets or numbers] position will be.It has 2 values: inside:It displays the list marker inside the content box means any property applied to the content will also take effect upon it. outside(default):It displays the list marker outside the content box means any property applied to the content will not take effect upon it.

list-style-image.

Explanation:

You can also use images and gifs as markers in list-style-type with the syntax: list-style-type: url("url of image")
